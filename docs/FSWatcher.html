Class: FSWatcher

FSWatcher()

Watches files & directories for changes. Emitted events: `add`, `addDir`, `change`, `unlink`, `unlinkDir`, `all`, `error` new FSWatcher() .add(directories) .on('add', path => log('File', path, 'was added'))
